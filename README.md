# Description

Application to compute the execution time of a software in a linux or macOS terminal

# How to install

```sh
# cp runtime $HOME/bin

# chmod +x $HOME/bin/runtime

# nano ~/.bashrc

export PATH = $PATH:$HOME/bin

# source ~/.bashrc
```

# How to run

```sh
# runtime <Linux_Executable_ApplicationName> <Enter>
```
